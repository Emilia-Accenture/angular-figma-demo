# Pipeline launcher

Prompt da usare quando non hai tutti i parametri pronti per `/figma-pipeline`.

```
Voglio generare una nuova schermata Angular. Aiutami a definire:
- nome (kebab-case)
- figmaNode (oppure userò screenshot allegato)
- specPath (opzionale, default docs/SPEC_<nome>.md)

Riepiloga i parametri e, su conferma, lancia /figma-pipeline.
```

Per uso diretto: `/figma-pipeline nome="..." figmaNode="..."`.
